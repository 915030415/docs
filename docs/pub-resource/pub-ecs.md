# 云服务器
**功能概述** 云服务器ECS提供安全可靠的弹性计算服务，支持多种实例规格和操作系统。

**主要功能**
- **实例管理**：创建、启动、停止、重启、删除云服务器
- **规格调整**：根据业务需求调整CPU、内存配置
- **系统管理**：支持多种Linux和Windows操作系统

**云实例特性**
- 支持包年包月和按量付费两种计费模式
- 若分配公网IP，带宽费用按量收费

**操作指南**

**1、创建云服务器**
- 选择"云服务器" → "云实例"
- 点击"创建实例"
- 配置实例规格、镜像、网络等参数
- 设置密码或密钥对
- 确认配置并创建

**2、管理云服务器**
- 在实例列表中查看所有服务器
- 支持批量操作和状态筛选
- 可查看详细的实例信息
